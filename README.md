O **Semeando o Futuro** é uma plataforma web interativa, responsiva e de página única (SPA - *Single Page Application*), desenvolvida para conscientizar a sociedade sobre a extrema importância da **agricultura familiar** no Brasil. 

O projeto apresenta dados estatísticos reais, os pilares da produção sustentável e conta com ferramentas interativas para aproximar o cidadão urbano da realidade do campo.

---

## 🚀 Funcionalidades Clave

* **Navegação Dinâmica (SPA):** Transição suave entre seções sem necessidade de recarregar a página, otimizando o desempenho e a experiência do usuário.
* **Design Moderno e Limpo:** Interface construída com **Tailwind CSS v4**, utilizando paletas de cores que remetem à terra e à natureza (tons de `stone` e `emerald`).
* **Plano de Fundo Tipográfico:** Efeito visual sutil no fundo da página reforçando palavras-chave do ecossistema agrícola.
* **Simulador de Impacto:** Ferramenta em tempo real construída em JavaScript que calcula a projeção anual do investimento em feiras locais e o número estimado de famílias apoiadas.
* **Formulário de Cadastro:** Área simulada para novos contribuintes com validação de dados e feedback visual de sucesso.
* **Responsividade Total:** Adaptado perfeitamente para visualização em smartphones, tablets e computadores, incluindo menu *mobile* dinâmico.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias modernas diretamente via CDN, eliminando a necessidade de etapas complexas de compilação ou instalação local:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **Tailwind CSS (v4):** Framework utilitário para estilização rápida, responsiva e moderna.
* **JavaScript (Vanilla):** Lógica de alternância de abas, cálculos matemáticos do simulador e manipulação do DOM.
* **Lucide Icons:** Biblioteca de ícones vetoriais leves e minimalistas para a interface.
